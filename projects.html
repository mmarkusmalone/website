Personal Projects

VADMap

VADMap is an interactive web application that transforms personal journaling into a rich, visual exploration of emotion. By combining AI-powered sentiment analysis with immersive 3D graphics, it helps users reflect on their emotional journeys over time through the psychological dimensions of Valence, Arousal, and Dominance (VAD). The frontend was built with Javascript/HTML/CSS. The backend was built with Python and Flask. The model was finetuned with HuggingFace and PyTorch.

Dance Partner

A web app that uses MediaPipe for real-time motion tracking: skeleton + aura effects over your webcam feed. Interactive buttons let you adjust colors, and WebGL is used for dynamic visual ambiance synced with music. Built with JS/HTML/CSS.

Scrap

A collage/remix tool for digital content: combine media (images, video, etc.) in creative ways. Helps user explore visual recombination. Built with HTML/CSS.

Dreamscape

A web app that extracts proper nouns from dream entries using an LLM, mapping co-occurring terms into interactive graph visualizations to reveal recurring dream themes.

Allies with Advice Website

I independently designed and launched Allies With Advice, a digital platform that connects mentors with pediatric arthritis patients through a supportive letter-exchange system. The site enhances accessibility and community engagement by providing a safe space for sharing experiences, encouragement, and guidance tailored to diverse user needs.
